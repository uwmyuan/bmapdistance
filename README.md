# bmapdistance
Inquiry distance and duration between locations with Baidu Map API 
使用百度地图API查询两个坐标点的路网距离和行程时间 
支持多种交通方式 
支持多点间查询
# Installation
```
# Github installation
install.packages("devtools")
devtools::install_github("uwmyuan/bmapsdistance")
```
